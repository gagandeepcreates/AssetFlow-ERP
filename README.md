# AssetFlow ERP

AssetFlow ERP is a robust Enterprise Resource Planning (ERP) system designed to streamline asset management, maintenance tracking, and organizational workflows.

## 🚀 Overview

AssetFlow ERP helps businesses manage their operational assets efficiently. From procurement to maintenance and reporting, this application provides a centralized platform for tracking the lifecycle of critical resources.

## 🛠 Project Structure

The project is structured to separate concerns between the frontend (user interface) and the backend (server logic).

```text
ASSETFLOW-ERP/
├── backend/            # Server-side API and database logic
├── frontend/           # Client-side UI and assets
│   ├── css/            # Style sheets
│   ├── js/             # Frontend application logic
│   ├── dist/           # Compiled/built files (ignored by Git)
│   └── src/            # Source code
├── .gitignore          # Files to be ignored by version control
├── package.json        # Dependencies and scripts
└── vite.config.js      # Frontend build configuration
```

## ⚙️ Key Features

- **Asset Tracking** — Monitor asset status, location, and assignments.
- **Maintenance Management** — Schedule and track maintenance tasks to reduce downtime.
- **Reporting** — Generate insightful reports on asset performance and utilization.
- **Organization Setup** — Manage users, departments, and roles.

## 🏗 Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript, Vite
- **Backend:** Node.js (assuming)
- **Tooling:** Git for version control

## 📦 Installation & Setup

**1. Clone the repository**

```bash
git clone https://github.com/your-username/assetflow-erp.git
cd assetflow-erp
```

**2. Install dependencies**

```bash
# Install root dependencies
npm install

# Navigate to frontend and install
cd frontend
npm install
```

**3. Run the development environment**

```bash
# From the root directory
npm run dev
```
